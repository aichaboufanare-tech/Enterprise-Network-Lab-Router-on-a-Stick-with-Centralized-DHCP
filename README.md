# Enterprise-Network-Lab-Router-on-a-Stick-with-Centralized-DHCP
This project simulates a small enterprise network using **GNS3** and Cisco IOU devices. It demonstrates VLAN segmentation, Inter-VLAN routing (Router-on-a-Stick), and centralized DHCP deployment using an Ubuntu server.  The lab is designed for CCNA–CCNP level practice.
